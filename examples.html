jQuery Toggler

Trigger hidden content with a single or a series of specific keystrokes.

Installation

  • Download jQuery Toggler: Github
  • Load jQuery
    • <script src="//ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js"></script>
  • Load Toggler
    • <script src="jquery.toggler.js"></script>
  • Have a piece of content you'd like to appear, which is presumably hidden by default
    • <div class="secret-content">CONTENT</div>
  • Register toggler behaviour on this DOM element.
    • <script type="text/javascript">
(function($) {
	$(document).ready(function() {
		$('.secret-content').toggler();	
	});
})(jQuery);
</script>

Example 1, No Options

By default, Toggler will use the Quake-visor inspired tilde (~) to trigger a slideUp/slideDown animation. Press tilde (~) twice within 2 seconds to activate it.

$('.example1').toggler();

Example 2, Konami Code

If you'd like to add the Konami code to your website, Toggler makes it quite simple.

Type out the Konami code (up, up, down, down, left, right, left, right, b, a) within 10 seconds to see it in action.

Example 2 
$('.example2').toggler({

	// up, up, down, down, left, right, left, right, b, a
	keyCode: [38,38,40,40,37,39,37,39,66,65],
	
	// 10 seconds
	duration: 10000
});

Example 3, Custom Toggle Animation

If you'd prefer to control the animation of the toggling, you can pass in your own callback function.

Here's how you'd change the slideUp/slideDown to use fadeIn/fadeOut instead. Press the `3` key twice to see it.

Example 3 
$('.example3').toggler({

	// 3, 3
	keyCode : [51,51],
	
	toggleOn : function(obj) {
		obj.fadeIn(1000);
	},
	toggleOff : function(obj) {
		obj.fadeOut(1000);
	}
});

Example 4, Making Your Own Key Combination

Toggler has a debug mode to help you make new key combinations. Turn it on and it'll print the keyCode matching the key you just pressed in the brower's console (like Firebug or the built in Chrome console).

Toggler.debugMode();
